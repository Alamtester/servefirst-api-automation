# Sreve First - Playwright API Automation

## Tech Stack
- Playwright (Javascript)
- Node.js

## Setup Instructions

1. Clone repository
2. Run: nmp install
3. Excecute: npx playwright test
4. View HTML Report: npx playwright show-report

## Test Coverage

1. Get all posts
2. Get single post
3. Create post
4. Update post
5. Delete post
6. Error handling
7. Data type validation
